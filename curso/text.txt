O que são cookies?

Um arquivo de texto que é enviado por um servidor web para o navegador de internet no momento em que o usuário acessa um endereço web, o arquivo é armazenado no disco rígido do dispositivo (computador ou smartphone).

Exemplos de informações coletadas através de cookies:
- Páginas e conteúdos acessados
- Sistema operacional
- Informações de localização
- Tipo de navegador
- Data e hora
- Informações do dipositivo usado
- Endereço de IP7


O que é a LGPD?

A Lei Geral de Proteção de Dados foi criada para o contexto brasileiro em 2018, é proposta para estabelecer princípios, direitos e obrigações relacionados a utilização de dados pessoais.

Princípios da LGPD para o tratamento de dados:
- finalidade
- livre acesso
- segurança
- prestação de contas
- adequação
- transparência
- prevenção da não discriminação
- necessidade
- qualidade dos dados
- responsabilização

Apenas deixe o usuário escolher

1. Teremos um modal de preferências, ele surge quando clicamos no botão “Gerenciar preferências de cookies”.
2. No mínimo dois botões:
Recusar todas opções de cookies
Confirmar minhas escolhas
3. Cada “categoria” de cookie deve ter uma descrição detalhada sobre o que é. Mas de forma simples, não desperdice palavras.
4. Evitar colocar cookies como obrigatórios, portanto cada item da lista deve ter um checkbox para ser selecionado.
